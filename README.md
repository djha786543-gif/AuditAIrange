# Run and deploy your audit portal

This contains everything you need to run the local audit tracker.

View your app in the browser after running it locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Paste your Gemini key into Settings on first launch — stored locally, never transmitted anywhere except google.com/gemini.
3. Run the app:
   `npm run dev`
