# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/c0ebaf9f-21b3-44ea-8411-c00d144ff28c

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Paste your Gemini key into Settings on first launch — stored locally, never transmitted anywhere except google.com/gemini.
3. Run the app:
   `npm run dev`
